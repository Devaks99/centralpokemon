# Central Pokemon

Bem-vindo à **Central Pokemon**! Este projeto é um site dedicado a explorar o mundo dos Pokémon, oferecendo informações sobre os Pokémon, jogos gratuitos e uma Pokédex interativa. A Central Pokemon tem como objetivo proporcionar uma experiência divertida e informativa para os fãs da franquia Pokémon, com acesso fácil a jogos e dados detalhados sobre cada Pokémon.

## Funcionalidades

- **Home**: Página inicial com uma breve introdução sobre o projeto.
- **Sobre**: Seção que detalha sobre cada Pokémon.
- **Jogos**: Acesso a jogos gratuitos baseados no universo Pokémon, como quebra-cabeças, labirintos e outros desafios.
- **Pokédex**: A Pokédex interativa com informações detalhadas sobre todos os Pokémon, incluindo tipos, ataques, evoluções, entre outros.

## Tecnologias Utilizadas

- **HTML**: Para estruturar as páginas e conteúdos do site.
- **CSS**: Para estilizar e dar o visual ao site, incluindo animações e responsividade.
- **JavaScript**: Para funcionalidades interativas no site, como animações e interações com o usuário.
- **Bootstrap**: Framework utilizado para o design responsivo e elementos UI prontos (como o carrossel de imagens).
- **Imagens**: Utilização de imagens do universo Pokémon para enriquecer a experiência visual.
